HTML

HTML(HyperText Markup Language)은 웹 페이지와 웹 애플리케이션을 구성하는 데 사용되는 마크업 언어입니다. HTML은 텍스트, 이미지, 링크, 비디오 등 다양한 요소를 웹 페이지에 배치하고, 이 요소들이 브라우저에서 어떻게 표시될지를 정의합니다.

HTML 문서는 구조적으로 태그(tag)를 사용하여 작성됩니다. HTML 태그는 요소의 시작과 끝을 나타내며, 웹 브라우저가 해당 태그를 해석하여 화면에 콘텐츠를 표시합니다.
